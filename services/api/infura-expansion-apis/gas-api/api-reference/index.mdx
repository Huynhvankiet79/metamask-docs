This section provides reference information for the various Gas REST APIs. Use the APIs to:

- [Get EIP-1559 gas prices](./gasprices-type2.mdx)
- [Get the base fee history (in GWEI)](./basefeehistory.mdx)
- [Get the base fee percentile (in GWEI)](./basefeepercentile.mdx)
- [Get the busy threshold for a network](./busythreshold.mdx)

## Supported API request formats

You can call the Gas APIs in two ways:

- **Use the API key only** - Add your [API key](../../../../dashboard/secure-an-api/api-key.md)
    as a path option.
- **Use the API key and API key secret** - Use basic authentication and specify the API key
    and [API key secret](../../../../dashboard/secure-an-api/api-key-secret.md).

<Tabs>
  <TabItem value="API key only" label="Use an API key only" default>

```bash
curl -X 'GET' 'https://gas.api.infura.io/v3/<API_KEY>/networks/1/suggestedGasFees'
```

  </TabItem>
  <TabItem value="API key and API key secret" label="Use an API key and API key secret" >

```bash
curl -X 'GET' -u <API_KEY>:<API_KEY_SECRET> 'https://gas.api.infura.io/networks/1/suggestedGasFees'
```
  </TabItem>
  </Tabs>
